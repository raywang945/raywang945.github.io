Towards Time-varying Music Auto-tagging Based on CAL500 Expansion

(09/2013 - present)

Music auto-tagging refers to the task of automatically assigning semantic labels such as genre, mood, and instrument to music so as to facilitate text-based music retrieval. Although significant progress has been made in recent years, relatively little research has been done to deal with semantic labels that are time-varying within the same track. Existing approaches and datasets usually assume that different fragments of a track share the same set of tags and thereby disregard tags that are time-varying or local in time. To address this issue, we present a new publicly available dataset for segment-level music auto-tagging. The dataset, which is called CAL500exp, is an enriched version of the well-known CAL500 dataset used for conventional track-level tagging.

  • Results were submitted to IEEE ICME 2014 (pending)

  • Academia Sinica
  • Teamwork with Ju-Chiang Wang and Yi-Hsuan Yang

Boosting OCR Accuracy Using Crowdsourcing

(08/2012 - 09/2013)

Book digitizing is an important work in preserving ancient heritages. However, digitizing books contains a series of labor-intensive works, and one of them is to verify optical character recognition (OCR) outcomes. We propose a crowdsourceable OCR verification method. Content holders are able to leverage the power of crowds to complete verification tasks and avoid content leakage by using our method.

  • Results were published in AAAI HCOMP 2013 [pdf]
  • Academia Sinica
  • Teamwork with Ming-Hung Wang

Implementation of Beam-forming

(09/2010 - 06/2011)

This project aimed to detect the direction of source of sound by the implementation of beam-forming. We implemented beam-forming using microphone arrays. The project had several steps, including buying basic IC components, welding circuits, Matlab simulation, and FPGA control.
